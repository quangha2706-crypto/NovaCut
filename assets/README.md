# Assets

This directory contains application assets and resources.

## Directory Structure

```
assets/
├── icons/          # Application and UI icons
├── images/         # Images and graphics
├── fonts/          # Custom fonts
└── styles/         # Style sheets and themes
```

## Icons

Place application icons here:
- `icon.png` - Main application icon (1024x1024)
- `icon.ico` - Windows icon
- `icon.icns` - macOS icon

UI icons should follow a consistent style and size (typically 16x16, 24x24, 32x32, 48x48).

## Images

Placeholder images, splash screens, and other graphical assets.

## Fonts

Custom fonts used in the application. Remember to check licensing for distribution.

## Styles

Qt style sheets (QSS) for theming the application.

## Notes

- Use SVG format for scalable icons when possible
- Optimize images for size
- Include attribution for any third-party assets
